# Weather Data Analysis Project

## Overview
This project demonstrates my ability to analyse real-world data, identify patterns, and communicate insights clearly.

I explored relationships between environmental variables (e.g. temperature and wind speed) using Python, performing data cleaning, exploratory analysis, and regression evaluation.

**For the main analysis, please view:** `6.2C.ipynb`

---

## Key Insight
Temperature alone was not a strong predictor of wind speed, highlighting the importance of analysing multiple variables rather than relying on assumptions.
